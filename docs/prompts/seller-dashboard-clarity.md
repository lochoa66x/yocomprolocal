# Seller Dashboard Clarity

## Goal

Make the seller dashboard first-time flow easier for non-technical sellers to
understand.

## Core Flow

The dashboard should clearly guide sellers through three actions:

1. Edit profile.
2. Add product.
3. Share store.

## Scope

- Simplify the first-run checklist from a broad setup list into a focused
  three-step path.
- Use human, Latin American Spanish that sounds practical and friendly.
- Keep seller actions obvious and tap-friendly on mobile.
- Put the first-run checklist before secondary shortcut/share sections.
- Keep existing product management, share kit, product status, edit, delete,
  and publish/draft controls intact.

## Out Of Scope

- Payments.
- Shipping.
- Buyer accounts.
- Admin approval.
- Major visual redesign.
